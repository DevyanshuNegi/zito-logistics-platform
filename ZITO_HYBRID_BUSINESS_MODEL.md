## 🎯 BUSINESS MODEL (HYBRID - CORRECTED)

ZITO operates in **TWO PARALLEL STREAMS**:

### Stream 1: Own Fleet Operations (Direct Logistics)
```
ZITO Owns:
✓ Trucks (FTL, PTL, Courier vehicles)
✓ Drivers (employed staff)
✓ Operations infrastructure

ZITO Executes:
✓ Bookings from customers
✓ Direct deliveries
✓ Full-stack logistics service

Revenue: 100% of delivery fees
Margin: Typical logistics (30-40%)
Model: Like Aramex, DHL, traditional operator
```

### Stream 2: Platform Network (Marketplace)
```
ZITO Enables:
✓ Third-party Transporters (own their trucks/drivers)
✓ Courier Companies (operate on ZITO platform)
✓ Agents (broker capacity)
✓ Warehouses & partners

Revenue: Commission 5-10% per transaction
Margin: Software economics (70-90%)
Model: Like Grab, Uber Freight, marketplace

Value:
- Transporters get better load matching
- Drivers earn more (jobs pushed to app)
- Customers get more options + better pricing
- ZITO earns commission on every match
```

## 📊 OPERATION MODEL (Hybrid Execution)

### Booking Flow

#### Customer Books Delivery
```
Customer opens ZITO app
Selects: FTL or PTL or Courier
System shows 3 options:

Option 1: ZITO Fleet
- Price: KES 1,500 (premium)
- ETA: 2 days
- Guarantee: 99.9% on-time
- Service: White glove

Option 2: Partner Network
- Price: KES 1,000 (competitive)
- ETA: 2-3 days
- Guarantee: Best-effort
- Service: Standard

Option 3: Blended (Auto-Select)
- Price: Automatic mix
- System chooses best option
- Based on: Price, availability, customer history

Customer selects → Booking created → Payment → Dispatch
```

#### ZITO Fleet Dispatch
```
→ Order goes to ZITO Operations team
→ System checks: Own truck availability, driver availability
→ If available: Assign ZITO truck + driver
→ Driver accepts on ZITO Partners app
→ Execution: Full ZITO control

Revenue: 100% (ZITO keeps all)
Margin: ~35-40% after costs
```

#### Partner Network Dispatch
```
→ Order goes to ZITO Platform
→ System broadcasts to partner network
→ Closest available partner vehicle bids/accepts
→ Transporter driver accepts on ZITO Partners app
→ Execution: Partner controls, ZITO coordinates

Revenue: 8% commission (ZITO gets 8%, partner gets 92%)
Margin: ~90% (no costs, just platform fee)
```
